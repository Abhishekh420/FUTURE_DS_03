# Marketing Funnel & Conversion Performance Analysis

## Project Objective
Analyze marketing funnel data to identify conversion drop-offs, evaluate channel performance, and discover opportunities to improve lead-to-customer conversion rates.

## Dataset
Marketing Funnel Analysis Dataset

## Tools Used
- Microsoft Power BI
- Power Query
- DAX

## Project Files
- Funnel_Analysis.pbix
- Funnel_Analysis_Data.xlsx
- Marketing_Funnel_Analysis_Report.pdf
- funnelAnalysisSC.png

## Key Analysis Performed
- Funnel stage analysis
- Conversion rate analysis
- Customer journey tracking
- Channel performance evaluation
- Funnel drop-off identification
- Marketing performance insights

## Key Findings
- Identified the stage with the highest customer drop-off.
- Measured conversion rates across funnel stages.
- Evaluated marketing channel effectiveness.
- Analyzed lead-to-customer conversion performance.

## Business Recommendations
- Improve low-performing funnel stages.
- Optimize marketing channels with higher conversion rates.
- Reduce customer drop-offs through targeted engagement.
- Enhance lead nurturing strategies.
